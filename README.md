# Dynobil.Case:
Merhaba Proje marka model ve ilanlardan oluşan basit bir crud uygulamasıdır. nTier mimaride yazılmış ve MVC olarak kullanıma sunulmuştur. Data Access için basit repository yazılmış ve data boyutu büyük olmadığı için async programlama kullanılmamıştır. programı çalıştırmadan önce Connection String'i kendinize uygun olan şekilde düzenlemeyi unutmayınız. Seed data verilerin json dosyadan okunması için ikinci kez çalıştırma gerekebilir. Verilerin düzgün bir şekilde aktarıldığından dashboard ekranında yer alan 71 1314 ve 3 sayılarından doğrulayabilirsiniz. teşekkür ederim
